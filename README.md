# AWS Architecture Learning Scenarios

Learn AWS Solutions Architect concepts through realistic, evolving business scenarios.

## How It Works

Each scenario starts with a **simple Day 1 MVP** and evolves naturally through:

- Growth pains
- Reliability requirements
- Scaling challenges
- Global expansion
- Cost optimization

Every architectural decision explains the **WHY** - mapping directly to SAA exam concepts.

## Getting Started

1. Pick a scenario
2. Start with Phase 1 (MVP)
3. Work through each phase sequentially
4. Focus on understanding the **business trigger** for each evolution
